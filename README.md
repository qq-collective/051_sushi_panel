## 概要

回転寿司のタッチパネルUIをモチーフにしたデザインパレット生成アプリ。
好きな寿司ネタを選ぶと、Claude APIがその寿司の**色・構造・質感**を3つのデザインレイヤーに変換して出力します。

「神は細部に宿る」をもじって——
**お江戸の魂は、デザインに宿る。**

neco_palette（QQ #047）の姉妹編。
Vivillon Code「デザインソムリエ」構想の **Section 02 — 寿司** 。

## ふざけてる形式で、真面目なデザイン理論を返す

寿司は完璧なデザインオブジェクトです。

| 寿司の要素 | デザインへの変換 |
|-----------|----------------|
| シャリ | 背景色・余白・ベース領域 |
| ネタ | メインカラー・ブランド印象 |
| 海苔 | 境界線・フレーム・ナビゲーション |
| わさび | CTA・通知・強調点 |
| タレ | グラデーション・影・深み |
| 筋・粒・格子 | テクスチャ・装飾パターン |

「好きなネタ」という直感的な問いが、そのままUIデザインの設計思想に変換されます。

## 3レイヤー出力（一の重・二の重・三の重）

### 🎨 一の重 — カラーリング
シャリ・ネタ・海苔・タレ・わさびの色構成を  
**カラーパレット＋使用比率**に変換。  
「シャリホワイトが何割か」という問いが、UIの余白設計に直結する。

### 📐 二の重 — レイアウト構成
楕円のシャリ・帯状の海苔・ドーム型のネタという物理的な構造を  
**UIのレイアウト型・余白・形状特徴**に変換。

### ✨ 三の重 — アクセント
わさびのポイントグリーン・サーモンの白い筋・いかの格子模様を  
**UI装飾要素**に変換。指輪やペンダントのように、全体を引き締める一点。

## 使い方

1. `sushi-panel.html` をブラウザで開く
2. Anthropic API キーを入力して「保存」
3. プロファイルを選ぶ（稚魚/中くらい/成魚 × 遊びの席/仕事の膳）
4. 好きなネタを一品タップ
5. 「このネタで握る」ボタンを押す
6. 一の重・二の重・三の重でデザイン仕様を確認する

---

## 技術仕様

- 単体 HTML ファイル（依存なし・ビルドステップなし）
- Anthropic API `claude-sonnet-4-20250514` 使用
- 全画像 base64 埋め込み済み（オフライン動作可）
- API キーは `localStorage` に保存
- システムプロンプトに素材色トークン・変換ルール表を固定
